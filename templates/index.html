📊 Explorando a Função Quadrática

f(x) = ax² + bx + c

📌 O que é uma Função Quadrática?

É uma função de grau 2. Seu gráfico sempre forma uma parábola.

Como cada coeficiente influencia:

  • a: Abre a parábola para cima (a > 0) ou para baixo (a < 0). Quanto maior |a|, mais "fechada" fica a parábola.
  • b: Desloca a parábola para a direita ou esquerda.
  • c: É o ponto onde a parábola corta o eixo y (f(0) = c).

🎯 Como usar esta ferramenta

1. Altere os valores de a, b e c nos campos acima.
2. Observe como o gráfico muda em tempo real.
3. Analise o formato, vértice, raízes e concavidade.

✍️ Exemplos Importantes

Teste estes valores:

  • a = 1, b = 0, c = 0 → x²
  • a = -1, b = 0, c = 9 → 9 - x² (máximo em y=9)
  • a = 1, b = -4, c = 3 → Tem duas raízes
  • a = 0.5, b = 0, c = -2 → Parábola mais aberta

📚 Exercícios para você resolver

1. Qual o valor de a para que a parábola abra para baixo?
2. Encontre os valores de a, b e c para que a parábola passe pelos pontos (0,0), (2,4) e (4,0).
3. Qual o vértice da função f(x) = x² - 6x + 5? (use o gráfico para conferir)
4. Para qual valor de c a parábola corta o eixo y em 8?
